# 소개
마인크래프트 OpenVanilla 서버에서 자체 개발되었던 기능의 일부입니다. 이러한 기능이 필요한 분들을 위해, 복잡한 설정 없이 바로 사용할 수 있는 기능을 모아 배포합니다.

# 사용권
저는 간단한 아이디어에서 시작했던 독창적인 기능이 마인크래프트를 더 다채롭게 만들 수 있기를 바랍니다. 이 문서를 포함하여 함께 배포된 플러그인은 '퍼블릭 도메인'으로 지적재산권을 포기함을 명시합니다. 누구나 언제든 어디에든 자유롭게 사용할 수 있습니다. 따라서 귀하는 개인적인 사용 이외에도 코드 디버깅 또는 리버싱을 포함한 모든 권한을 가집니다. 수정하여 재배포할 수 있고, 그 자체로도 재배포할 수 있습니다. 단, 저작인격권은 영구적으로 보존됩니다. 자세한 사항은 아래 바로가기를 참고해주세요.

https://ko.wikipedia.org/wiki/퍼블릭_도메인

# 모든 파일에 적용되는 설명
 * 대다수의 프로그램은 Paper API에 종속적이며, Paper 환경에서만 작동합니다.
 * 이 파일은 모든 환경에서 작동하도록 설계되지 않았습니다. 일부 프로그램에는 비권장(Deprecated) 코드가 포함되어 있을 수 있으며, 본인은 개발에 사용된 API 또는 자바 개발 환경 지원 종료에 따른 이후 발생하는 문제에 대한 모든 사후 지원을 제공하지 않습니다.

# 각 파일의 설명

## Chairs
계단으로 분류된 모든 블록에 우클릭 시 앉기 기능을 추가합니다. 설정 파일을 제공하지는 않으나, 업데이트 이후에도 모든 계단 블록이 버킷 API 분류에 따라 자동으로 지원됩니다. 사소한 디테일: 정면, 대각선 방향 모두 올바른 위치에 앉은 것으로 보입니다.

## Hat
서바이벌 모드에서 모자 슬롯에 모든 아이템을 장착할 수 있습니다. 일반 아이템을 모자 슬롯에 그냥 클릭해 놓아보세요. 참고: 터치 환경에서의 테스트는 이루어지지 않았습니다.

## Invite
초대 명령어(/invite)를 추가합니다. Bukkit API 권한(LuckPerms 등) 설정에 따라 초대를 받을 수 있는지, 초대를 보낼 수 있는지 등을 설정할 수 있습니다. 참고: 권한 오류 메시지가 '이 명령어는 오버월드에서만 사용할 수 있습니다'로 표시됩니다. 코드 리버싱으로 수정할 수 있습니다.

## PlayerVehicle
플레이어 머리 부분을 우클릭하면 해당 플레이어의 머리 위로 탈 수 있는 기능을 추가합니다. 아래 있는 플레이어가 앉으면 위에 있는 플레이어가 아래로 떨어집니다.

## TeleportPlus
순간이동 관련 명령어를 추가합니다. 다른 세계로 이동하거나, 청크 좌표 단위로 이동하거나, 보고 있는 방향으로 이동하거나, 맨 위 블록으로 이동하는 명령어가 추가됩니다.

## Waypoint
집 위치(/getspawnpoint) 및 순간이동 이전 위치(/previouslocation) 및 최근 사망 위치(/whyamidead) 표시 명령어를 추가합니다. 권한 설정에 따라 다른 플레이어의 웨이포인트를 확인할 수도 있습니다.
